# 广州地铁报站模拟器

## 安装项目 Project setup
```bash
yarn install

// 或使用npm

npm install
```

### 开发 Compiles and hot-reloads for development
```bash
yarn serve

// 或使用npm

npm run serve
```

### 编译输出 Compiles and minifies for production
```bash
yarn build

// 或使用npm

npm run build
```
